# Kakao Tools 본선 확장 계획

본선 진출 시 약 한 달의 추가 개발 기간에 공식 Widget JSON 스펙을 적용합니다.

## Widget 경험

- 이미지 선택 또는 촬영
- OCR 결과를 왼쪽 상대방·오른쪽 사용자 말풍선으로 미리보기
- 잘못 인식된 텍스트를 탭해 수정
- 원문과 번역을 접어서 비교
- 발신 말풍선에서 교정본과 번역본 중 복사할 문장을 선택
- 처리 중, 부분 실패, 지원하지 않는 언어, 번역 fallback 상태 표시

## 서버 확장

- OCR 작업을 빠른 작업 생성과 상태 조회로 분리해 모든 tool 응답을 3초 안에 유지
- Widget 전용 최소 structured content와 output schema 정의
- 이미지 업로드용 서명 URL과 짧은 만료 시간 적용
- 사용자가 명시적으로 저장하지 않는 한 이미지와 원문 즉시 폐기
- 언어 모델 패키지를 수요 기반으로 추가하고 언어별 품질 회귀 테스트 운영

Tool 이름에는 서비스명이나 금지 문자열을 넣지 않고, PlayMCP가 부여하는 prefix를 그대로 사용합니다.
