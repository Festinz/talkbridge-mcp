## 최종 포지셔닝

**TalkBridge는 대화 캡처의 말풍선 방향을 읽고, 받은 말의 이해와 보낼 답장의 교정·번역을 한 번에 이어주는 역할 인식형 대화 브리지입니다.**

차별화 축:

1. 좌측 상대방과 우측 사용자를 구분해 번역 방향을 자동 결정합니다.
2. 수신은 내 언어 번역, 발신은 맞춤법 교정 후 상대 언어 번역이라는 비대칭 파이프라인을 제공합니다.
3. 한 문장이 아니라 여러 말풍선의 순서와 역할을 유지합니다.
4. OpenAI나 유료 번역 API 없이 로컬 OCR·NLLB 모델로 동작합니다.
5. 이미지·원문을 저장하지 않고 fallback을 명시해 결과를 과장하지 않습니다.
6. PlayMCP 도구는 빠른 텍스트·발화 배열 처리로 분리해 응답 시간 기준을 지킵니다.

## 평가 기준 대응

- 창의성: OCR 자체가 아니라 말풍선 위치를 번역 방향으로 사용하는 역할 인식 흐름
- 편의성: 복사·붙여넣기 한 문장과 대화 캡처 여러 문장을 모두 지원하고 바로 보낼 답장을 제공
- 안정성: Stateless Streamable HTTP, 짧은 tool surface, 요청 제한, timeout, 명시적 fallback
- 파급력: 해외 친구, 다문화 가족, 유학생, 여행, 글로벌 업무 등 메신저가 쓰이는 모든 일상으로 확장 가능
